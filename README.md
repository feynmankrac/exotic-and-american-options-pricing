## Overview
This project focuses on the pricing of exotic and American options using numerical methods implemented in VBA.

## Scope
- Path-dependent exotic options
- American-style options
- Numerical pricing methods

## Methodology
- Monte Carlo simulation
- Binomial tree approach
- Calibration and pricing gap analysis

## Implementation
- VBA (Excel)
- Modular pricing framework
- Numerical validation

## Planned extension
A Python implementation is planned to extend the framework and improve scalability and performance.

## Files
- `src/`: VBA pricing engines
- `report/`: methodology and results
